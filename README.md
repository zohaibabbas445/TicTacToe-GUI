## Tic-Tac-Toe (Python GUI)

A two-player Tic-Tac-Toe game with a graphical interface using Python and Tkinter.
I originally wrote this game in C++ as a console application. I converted it to Python to learn GUI development and object-oriented programming.

## Features
Two players (X and O) take turns on a 3x3 grid.
Automatically detects win (row, column, or diagonal) or draw.
"New Game" button resets the board instantly.

## Tech Stack
Python 3
Tkinter (built-in)

## How to Run
Make sure Python 3 is installed. Then open a terminal in the project folder and run:
python main.py
